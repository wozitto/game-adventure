# Game Adventure

うまく障害物をよけながら、さまざまなゲームを集めるアドベンチャーゲームです。

![ゲームのデモのスクリーンショット](./images/demo.png)

## Getting Started

以下のコマンドを実行すると、[http://localhost:3000](http://localhost:3000)でアプリケーションが立ち上がります。

```bash
docker-compose up --build
```

### Development

開発サーバーを起動するには、以下のコマンドを実行してください。

```bash
npm run dev
```
